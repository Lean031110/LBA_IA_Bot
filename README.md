# LBA IA Bot 🤖

Bot de Telegram con capacidades de IA y moderación automática.

## 🌟 Características

- 🤖 IA adaptativa para respuestas automáticas
- 🔍 Búsqueda web y Wikipedia integrada
- 👥 Moderación automática de grupos
- 📊 Panel de estadísticas en tiempo real
- 💾 Sistema de respaldo automático
- 🔄 Aprendizaje continuo

## 🛠 Requisitos

- Python 3.8+
- pip
- Telegram Bot Token
- Conexión a Internet

## 🚀 Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Lean031110/lba_ia_bot.git
   cd lba_ia_bot
   ```

2. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```

3. Configurar el token del bot de Telegram:
   - Crea un archivo `.env` en el directorio raíz del proyecto.
   - Añade tu token de Telegram en el archivo `.env`:
     ```
     TELEGRAM_TOKEN=tu_token_aqui
     ```

## 🏃 Uso

Para iniciar el bot, ejecuta:
```bash
python main.py
```
